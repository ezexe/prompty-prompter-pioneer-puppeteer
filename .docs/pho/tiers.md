# φ-register Rosetta Stone

Five domains, one structure — the translation dictionary.

## The constraint

| Structural concept              | φ-register                     | CSD accelerators                                     | QDI circuits                                                 | Neural CA / rewriting                  | Coding theory                            |
| ------------------------------- | ------------------------------ | ---------------------------------------------------- | ------------------------------------------------------------ | -------------------------------------- | ---------------------------------------- |
| No two adjacent elements active | No consecutive 1s (Zeckendorf) | No adjacent nonzero signed digits (canonic recoding) | Mutual exclusion between adjacent rails (dual-rail encoding) | Neighborhood constraint on update mask | Forbidden word '11' in golden mean shift |
| The generator of the constraint | M = [[1,1],[1,0]], φ² = φ+1    | Booth recoding scan of adjacent bit pairs            | Muller C-element gate topology                               | Learned local update kernel            | Transition matrix of the shift space     |

## The process

| Structural concept                              | φ-register                                   | CSD accelerators                                       | QDI circuits                                  | Neural CA / rewriting                              | Coding theory                                    |
| ----------------------------------------------- | -------------------------------------------- | ------------------------------------------------------ | --------------------------------------------- | -------------------------------------------------- | ------------------------------------------------ |
| Local rewrite that resolves a violation         | 011 → 100 (carry rule)                       | Replace digit pair (1,1) with (−1,0,1) or equivalent   | Handshake: req/ack exchange between neighbors | Cell applies update function to local neighborhood | Context-free substitution rule in the transducer |
| Intermediate state that violates the constraint | Digit 2, or adjacent 1s before normalization | Non-canonic representation (adjacent nonzeros present) | Spacer / invalid codeword in dual-rail        | Unstable cell configuration                        | Non-prefix-free codeword; ambiguous parse        |
| Propagation direction                           | Bidirectional: +1 left, −2 right (Fenwick)   | Left-to-right scan; borrow propagates right            | Completion tree: leaves → root (upward)       | Neighborhood radius determines propagation extent  | Berstel transducer: left-to-right, 4 states      |

## The result

| Structural concept           | φ-register                                      | CSD accelerators                                   | QDI circuits                                          | Neural CA / rewriting                       | Coding theory                                            |
| ---------------------------- | ----------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------- | ------------------------------------------- | -------------------------------------------------------- |
| Global consistency achieved  | Zeckendorf normal form                          | Canonic Signed Digit form (minimal Hamming weight) | All handshakes complete; output valid                 | Stable fixed point of iterated updates      | Valid prefix-free encoding; delimiter '11' placed        |
| Completion signal            | No cell contains 2; no adjacent 1s              | All digit pairs satisfy adjacency check            | Completion tree output fires (matched rising/falling) | State delta < ε (convergence criterion)     | '11' delimiter marks end-of-codeword                     |
| Error / corruption detection | Cassini identity: F(n-1)·F(n+1) − F(n)² = (−1)ⁿ | Non-canonic digits in output (should never appear) | Dual-rail: both rails high = error (1-of-2 violation) | Divergence or oscillation = failed training | Constraint violation mid-stream → resync within 3 tokens |

## Tier 1 — Established and deployed

Results with working hardware, production systems, or mature publication records.

| Intersection                       | What exists                                                                                                                                                                                                                                     | Key literature                                                                                                                                           |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CSD multipliers for CNNs           | Canonic Signed Digit approximation of CNN filter weights. 77% logic area reduction, 5.63% top-1 accuracy drop on VGG-16/ImageNet, no retraining. The adjacency constraint (no adjacent nonzero digits) is structurally identical to Zeckendorf. | Riaz et al., "CAxCNN," IEEE Access 2020; MA4C, ScienceDirect 2022 (4.2× FPGA logic reduction); RNS-CSD CNN accelerator at 22nm (1.54× energy efficiency) |
| CSD in general processors          | CSD encoding in a RISC-V pipeline achieves 2.41× clock frequency increase over standard binary, >20% from the encoding itself.                                                                                                                  | Reichenbach et al., "CSD-Driven Speedup in RISC-V Processor," Springer 2025; companion project "RISC-V3"                                                 |
| Carry-save in TPU systolic arrays  | Google TPU patent describes carry-save adders (redundant binary) within MAC units, passing carries between units in partially redundant form. Redundant number arithmetic inside the most deployed AI accelerator.                              | US Patent US20230015148A1, Google 2023                                                                                                                   |
| Ternary weight networks            | Weights constrained to {−1,0,+1} — structurally identical to balanced ternary digits. 16× model compression, minimal accuracy loss. Mature field with memristor implementations.                                                                | Li & Liu, "Ternary Weight Networks," arXiv:1605.04711, 2016; Zhu et al., "Trained Ternary Quantization," ICLR 2017; memristor TWN, IEEE 2024–2025        |
| QDI neuromorphic circuits          | Quasi-delay-insensitive design where local handshaking settles to global consistency — structurally identical to Fibonacci normalization. | Fibonacci lattice sampling         | Golden-angle-based Fibonacci spiral as optimal quasi-uniform spherical point distribution. | Golden mean shift capacity         | Channel capacity of binary sequences with no consecutive 1s = log₂(φ) ≈ 0.694 bits/symbol.
